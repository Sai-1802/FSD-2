# Screenshots Folder

Add 2-3 screenshots here that show:
1. Home page with products and theme
2. Cart page demonstrating Redux

## Screenshot Guidelines
- Use descriptive names: `home.png`, `cart.png`, `analytics.png`
- Include UI elements in the screenshots
- Show both light and dark modes if possible
- Dimensions: 1200x800px or larger

## How to Add Screenshots

1. Open your project in development mode: `npm run dev`
2. Take screenshots of different pages
3. Save them as PNG files in this folder
4. Update the README.md with screenshot references
